## 游戏介绍
在《雷霆战机》中，玩家操控一架飞机与敌方的源源不断产生的飞机战斗，一边要发射炮弹尝试击中敌方的飞机，一边又要注意自身飞机的身位来努力躲避敌方飞机发射的炮弹。通过击败敌方的飞机来获得分数，并坚持更多的回合数，击落更多敌机，获得更多的分数。
